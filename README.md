# power-picker
🌈 Web based color picker (Eye dropper) PowerPicker 🌤️

Webブラウザだけで画面に表示している画像の色情報を取得します。ScreenCaptureAPIでブラウザでないアプリケーションの色情報の取得も可能です。

[試してみる](https://picker.anyfrog.net/)
